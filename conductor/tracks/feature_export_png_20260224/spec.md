# 功能規格 - 匯出個別 PNG 圖片

## 1. 目標
讓使用者能夠將輸入的文字列表轉換為多個獨立的 PNG 圖片檔案，方便用於其他設計軟體或單獨列印。

## 2. 使用者流程
1. 使用者在 GUI 中輸入文字內容（每行一個）。
2. 使用者點擊「匯出 PNG 圖片」按鈕。
3. 彈出資料夾選擇視窗，由使用者指定儲存路徑。
4. 程式開始產生 QR Code。
5. 程式將每個 QR Code 儲存為 PNG：
    - 檔名：以二維碼內容命名（需處理非法檔名字元）。
    - 格式：PNG。
    - 參數：採用目前 GUI 設定的尺寸與容錯率。
6. 完成後彈出成功提示。

## 3. 異常處理
- **非法字元**：如果二維碼內容包含 Windows 不允許的檔名字元（如 `\ / : * ? " < > |`），需自動替換為底線 `_`。
- **重複檔名**：若內容相同導致檔名重複，需加上序號避免覆蓋（如 `text.png`, `text_1.png`）。
- **取消儲存**：若使用者在選擇路徑時取消，應終止流程。
